# Product Vision

POTOKI is a calm local context memory tool for technical users in restricted or privacy-sensitive environments. It is meant for returning to work after hours or days and quickly remembering:

- what a potok is about,
- what happened recently,
- which decisions were made,
- what is parked,
- which small return notes can be continued,
- which notes and local artifacts belong to the context.

It is deliberately not a task manager, Kanban board, time tracker, analytics dashboard, or project management suite. The product shape is closer to a local memory stream plus an operational notebook.

The core entity is the **potok**. A potok holds durable context, while timeline entries hold the memory of that context over time.

POTOKI should feel quiet, local, and continuity-focused. The interface should help the user return to thoughts without adding management pressure, productivity scoring, workflow ceremony, or cloud dependency.
