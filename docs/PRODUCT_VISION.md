# Product Vision

POTOKI is a calm local context memory tool for technical users in restricted environments. It is meant for returning to work after hours or days and quickly remembering:

- what the stream is about,
- what happened recently,
- which decisions were made,
- what is parked,
- which small resume notes can be continued,
- which notes and local artifacts belong to the stream.

It is deliberately not a task manager, Kanban board, time tracker, analytics dashboard, or project management suite. The product shape is closer to a developer cockpit plus operational notebook.

The core entity is the Workstream. A workstream holds durable context, while timeline entries hold the stream's memory over time.
